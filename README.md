# إنبات القابضة | INBAT HOLDING

موقع إلكتروني تعريفي احترافي لشركة زراعية قابضة متخصصة في تقديم حلول زراعية متكاملة وذكية.

## المميزات

- ✅ تصميم حديث واحترافي متوافق مع الهوية البصرية
- ✅ دعم كامل للغة العربية (RTL)
- ✅ متجاوب مع جميع أحجام الشاشات (Responsive)
- ✅ محسّن لمحركات البحث (SEO Friendly)
- ✅ أداء عالي وسرعة تحميل ممتازة
- ✅ كود نظيف وقابل للتوسعة

## التقنيات المستخدمة

- **Next.js 14** - Framework React مع App Router
- **TypeScript** - للكتابة الآمنة للكود
- **Tailwind CSS** - لتصميم سريع ومرن
- **Lucide React** - للأيقونات الحديثة
- **Tajawal Font** - خط عربي احترافي

## الألوان

- **Dark Green**: `#1b4332`
- **Light Green**: `#a2c95e`
- **Off White**: `#fffdfe`

## التثبيت والتشغيل

### المتطلبات

- Node.js 18+ 
- npm أو yarn

### خطوات التشغيل

1. تثبيت المتطلبات:
```bash
npm install
```

2. تشغيل المشروع في وضع التطوير:
```bash
npm run dev
```

3. افتح المتصفح على:
```
http://localhost:3000
```

### البناء للإنتاج

```bash
npm run build
npm start
```
